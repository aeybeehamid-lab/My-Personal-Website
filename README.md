# Abdulhamid Inuwa — Personal Website

React + Vite portfolio site. **All your text and links live in one file:**

👉 **`src/content/siteContent.js`**

Full guide: **`src/content/HOW_TO_EDIT.md`**

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually http://localhost:5173).

## Add your files

| File | Put here |
|------|----------|
| Profile photo | `public/images/profile.jpg` |
| CV PDF | `public/cv/resume.pdf` |
| Project photos | `public/images/projects/` (match names in `siteContent.js`) |

## Build for production

```bash
npm run build
npm run preview
```

Deploy to Vercel — see **`docs/DEPLOY.md`** for step-by-step instructions.

**PowerShell blocking npm?** Run: `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser`
